# heck CLI

```shell-session
$ echo 'This is a heck CLI' | heck
this-is-a-heck-cli

$ echo 'This is a heck CLI' | heck -t shoutysnake
THIS_IS_A_HECK_CLI

$ heck -h # to show usage
```

## Install

```shell-session
$ cargo install --git https://vcs.gemmaro.mydns.jp/git/gemmaro/heck-cli.git
```
